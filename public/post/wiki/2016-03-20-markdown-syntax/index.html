{: rules=“groups”}

注意： {: rules="groups"}，这里的rules可以填写下面的值：

值 描述
none 没有线条。
groups 位于行组和列组之间的线条。
rows 位于行之间的线条。
cols 位于列之间的线条。
all 位于行和列之间的线条。

{: rules=“all”}

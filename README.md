### markdown 文章定义格式

```

	layout: post					#post样式
	title: 标题					    #标题
	description: "简介"			    #简介
	updated: 2015-11-5 17:17:00		#修改时间
	date: 2015-08-18 15:07:19 #发布时间
	tags: [标签1, 标签2]		#标签
	comments: true			#是否可用评论
	photos: #图片
 	 - img/golang_import.jpg

```

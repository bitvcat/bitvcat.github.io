你来到这个页面，通常有两个原因。

## 一、链接错误

原因：博客搬迁造成的旧链接失效。

例子：《vim配置以及使用vundle管理vim插件》的链接之前是

```
http://domicat.me/2015/07/28/2015-7-28-vim-config-and-plugin/
```

从hexo迁移到jekyll后，新的链接是

```
https://domicat.me/2015/07/28/vim-config-and-plugin/
```


**解决办法：** 在「[分类](/categories/)」中找。

## 二、维度攻击

不明力量入侵，你想要访问的页面被摧毁了。
